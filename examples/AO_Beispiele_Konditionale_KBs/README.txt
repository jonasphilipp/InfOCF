Die einzelnen Ordner (z.B. A_Saeugetiere4) beinhalten die Daten zur jeweiligen Wissensbasis:

1)Einen Ordner mit den Wissensbasis-Daten (z.B. KB_Saeugetiere4). Darin befinden sich:
		a) Ein InfOCF-Input File im .cl-Format (z.B. kb_Saeugetiere4.cl)
		b) Ein Text-File mit Beschreibungen der Variablen und Konditionale (z.B. readme_kb_Saeugetiere4.txt)

2)Einen Ordner mit den Anfragekonditionalen an die Wissensbasis (z.B. Queries_Saeugetiere4). Darin befinden sich:
		a) Ein InfOCF-Input File im .cl-Format (z.B. queries_Saeugetiere4.cl)
		b) Ein Text-File mit Beschreibungen der Variablen und Konditionale (z.B. readme_queries_Saeugetiere4.txt)
